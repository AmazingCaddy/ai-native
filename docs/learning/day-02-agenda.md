## 回溯规则

1. 当前问题的简短追问继续记录在当天 notes 中。
2. 跨主题、需要详细解释或可能跨天继续的追问，创建 `DISC-NNN`。
3. 展开讨论结束后，返回最近的未完成主问题。
4. 每日结束时记录延期问题和下一次继续位置。
5. 首轮解释后将当前问题标记为 `待追问`，只有学习者明确继续才移动游标。
