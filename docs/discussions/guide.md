# 展开讨论说明

这里保存课程主线之外值得独立追踪的知识讨论。讨论可以源于任何一天，也可以跨天继续。

## 何时创建独立讨论

满足以下任一条件时创建 `DISC-NNN`：

- 问题跨越多个课程主题
- 需要多个推理步骤、案例或图表解释
- 当天无法得出稳定结论
- 后续需要实验、查证或再次讨论
- 结论值得整理成独立复习材料

一句话澄清或当前问题的直接追问，继续保留在当天学习记录中即可。

## 状态

- `待讨论`：已经记录，尚未展开。
- `讨论中`：正在形成解释或验证结论。
- `待整理`：讨论已有阶段性结论，尚未提炼到复习材料。
- `已整理`：结论已经进入复习材料。
- `延期`：明确推迟，并记录恢复条件。

所有讨论先登记到[讨论索引](./index.md)，再使用[讨论线程模板](./thread-template.md)创建独立文件。
