# 双轨学习材料设计

## 目标

将学习过程和复习材料分开保存：学习记录负责保留真实的问答与纠错轨迹，复习材料负责提供经过整理的知识内容。两者按课程天数一一对应。

## 文件结构

- `docs/learning/day-XX-notes.md`：问题、原始回答、判断、讲解和掌握状态。
- `docs/review/day-XX-review.md`：知识结构图、精选问题、相对标准答案、易错点和记忆句。
- `docs/learning/README.md` 与 `docs/review/README.md`：分别说明两类材料的用途和维护规则。

## 同步规则

每轮知识问答先写入当天学习记录。讲解完成后，将稳定、可复用的知识提炼到当天复习材料。尚未作答的问题只进入学习记录，不提前在复习材料中公布答案。

“相对标准答案”表示当前课程范围内足够准确的参考答案，并非唯一措辞。遇到依赖场景或存在取舍的问题，答案必须写明条件和边界。

复习材料优先使用紧凑表格和 Mermaid 图表达关系，不重复保存学习者的错误答案。每日结束时检查两个文件是否同步，并通过站点构建验证 Markdown 和 Mermaid 内容。
